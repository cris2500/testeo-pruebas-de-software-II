# testeo-pruebas-de-software-II
descripción:
Se trata del proyecto final de pruebas de software II, el cual es el testeo de una api y de una aplicación web
las cuales se efectuaron en el segundo semestre del 2025.
Se implementaron postman y cypress
Intrucciones 
se debe tener node js y postman intalados
con visual studio code puede abrir una terminal y correr este comando en la raiz del proyecto
npx cypress open y el proyecto iniara en cypress 
en postman solo deben de importar el ambiente de pruebas y la colección
